# Assign a Role

Give the AI a persona for better context.

## The Power of "Act as..."

Starting with "Act as..." or "You are..." gives the AI a framework for how to respond.

## Examples

**Marketing:**
```
Act as a friendly marketing copywriter. Write Instagram captions for a small coffee shop.
```

**Technical:**
```
You are a patient programming tutor explaining concepts to beginners. Explain what a variable is.
```

**Creative:**
```
You are a haiku poet. Write 5 haikus about rain.
```

## Why This Works

- Sets the knowledge level
- Defines the communication style
- Gives relevant context
- Frames the response appropriately

## Real-World Example

**Without role:**
```
Write a summary of this article.
```

**With role:**
```
You are a busy executive assistant. Write a summary of this article that I can read in 30 seconds. Highlight only the actionable takeaways.
```

The role tells the AI to focus on what's useful for a busy person.

## Best Roles to Try

- "Act as a [profession]"
- "You are a [role] with [years] experience"
- "Pretend you are [persona]"
- "As an expert in [field]"

Experiment! Different roles give very different results.
