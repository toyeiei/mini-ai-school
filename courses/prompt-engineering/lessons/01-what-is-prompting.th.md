# Prompting คืออะไร?

นึกภาพ AI เป็นผู้ช่วยที่รู้มากแต่อ่านใจคุณไม่ออก คุณต้องบอกเขาตรงๆ ว่าต้องการอะไร

Prompting คือศิลปะของการถาม AI เพื่อให้ได้ผลลัพธ์ที่ดีที่สุด

## ทำไมสำคัญ?

Prompt ที่ดีได้คุณ:
- คำตอบที่เป็นประโยชน์ ไม่ใช่คำตอบทั่วไป
- ลดการถามตอบซ้ำ
- งานเสร็จจริงๆ

Prompt ที่คลุมเครือได้คุณ:
- คำตอบทั่วไป
- คำถามตามมา
- เสียเวลา

## ตัวอย่างง่ายๆ

**Prompt ไม่ดี:**
```
Write about dogs.
```

**Prompt ดี:**
```
Write a 200-word introduction about golden retrievers for a first-time dog owner. Include: temperament, exercise needs, and grooming requirements. Tone: friendly and encouraging.
```

เห็นความแตกต่างไหม? อันที่สองบอก AI ว่า อะไร เท่าไหร่ และอย่างไร

## แนวคิดหลัก

คุณเป็นผู้เชี่ยวชาญเรื่องสิ่งที่คุณต้องการ AI เป็นผู้เชี่ยวชาญเรื่องวิธีทำ งานของคุณคือสื่อสารให้ชัด

## สิ่งที่คุณจะเรียน

เมื่อจบคอร์สนี้:
- เขียน prompts ที่ชัดเจนและเฉพาะเจาะจง
- ใช้ตัวอย่างเพื่อนำทางผลลัพธ์
- มอบหมายบทบาทเพื่อบริบทที่ดีขึ้น
- แบ่งงานซับซ้อนเป็นขั้นตอน
- ปรับปรุงเพื่อให้ได้สิ่งที่ต้องการ

เริ่มกันเลย!
