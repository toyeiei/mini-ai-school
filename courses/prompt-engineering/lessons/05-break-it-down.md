# Break It Down

Complex tasks need simple steps.

## Why Break Things Down?

A single prompt asking for a full business plan often produces a generic, shallow result.

But asking for each section separately gives you detailed, thoughtful content.

## The Step Method

Instead of:
```
Write a complete business plan.
```

Try:
```
1. First, help me identify my target customer. Who is most likely to buy this product?
2. Then, what are the top 3 problems my product solves for them?
3. Next, how should I price this? Consider similar products on the market.
4. Finally, write a one-paragraph summary combining all of the above.
```

## When to Use This

- **Planning documents:** Do sections one at a time
- **Code:** Get the structure first, then implement each part
- **Research:** Gather info in stages before synthesizing
- **Creative work:** Draft outline, then expand each section

## Real Example

**One big prompt:**
```
Write a complete website for my bakery.
```

**Broken down:**
```
1. What pages does a bakery website typically need?
2. Write the homepage hero text for a family-owned bakery emphasizing homemade and local ingredients.
3. Create a menu structure with 5 categories.
4. Write an "About Us" paragraph for a family bakery that started in 1995.
```

## The Magic Phrase

Use "First... Then... Next... Finally..." to guide the AI through your process step by step.
