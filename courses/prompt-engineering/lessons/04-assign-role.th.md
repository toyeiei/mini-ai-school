# มอบหมายบทบาท

ให้ AI เป็นตัวตนเพื่อบริบทที่ดีขึ้น

## พลังของ "ทำหน้าที่เป็น..."

เริ่มต้นด้วย "Act as..." หรือ "You are..." ให้ AI กรอบสำหรับวิธีตอบ

## ตัวอย่าง

**การตลาด:**
```
Act as a friendly marketing copywriter. Write Instagram captions for a small coffee shop.
```

**เทคนิค:**
```
You are a patient programming tutor explaining concepts to beginners. Explain what a variable is.
```

**สร้างสรรค์:**
```
You are a haiku poet. Write 5 haikus about rain.
```

## ทำไมถึงได้ผล

- กำหนดระดับความรู้
- กำหนดสไตล์การสื่อสาร
- ให้บริบทที่เกี่ยวข้อง
- จัดกรอบการตอบอย่างเหมาะสม

## ตัวอย่างใช้จริง

**ไม่มีบทบาท:**
```
Write a summary of this article.
```

**มีบทบาท:**
```
You are a busy executive assistant. Write a summary of this article that I can read in 30 seconds. Highlight only the actionable takeaways.
```

บทบาทบอก AI ให้โฟกัสสิ่งที่เป็นประโยชน์สำหรับคนยุ่ง

## บทบาทยอดนิยม

- "ทำหน้าที่เป็น [อาชีพ]"
- "คุณเป็น [บทบาท] ที่มีประสบการณ์ [จำนวน] ปี"
- "สมมติว่าคุณเป็น [บุคลิก]"
- "ในฐานะผู้เชี่ยวชาญด้าน [สาขา]"

ทดลอง! บทบาทต่างกันให้ผลลัพธ์ต่างกันมาก
