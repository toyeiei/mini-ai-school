# มอบหมายบทบาท

ให้ AI เป็นตัวตนเพื่อบริบทที่ดีขึ้น

## พลังของ "ทำหน้าที่เป็น..."

เริ่มต้นด้วย "Act as..." หรือ "You are..." ให้ AI กรอบสำหรับวิธีตอบ

## ตัวอย่าง

**การตลาด:**
```
ทำหน้าที่เป็นนักเขียนคัดลอกการตลาดที่เป็นมิตร เขียนคำบรรยาย Instagram สำหรับร้านกาแฟเล็กๆ
```

**เทคนิค:**
```
คุณเป็นติวเตอร์การเขียนโปรแกรมที่ใจเย็นอธิบายแนวคิดสำหรับมือใหม่ อธิบายว่า variable คืออะไร
```

**สร้างสรรค์:**
```
คุณเป็นกวีไฮกุ เขียนไฮกุ 5 บทเกี่ยวกับฝน
```

## ทำไมถึงได้ผล

- กำหนดระดับความรู้
- กำหนดสไตล์การสื่อสาร
- ให้บริบทที่เกี่ยวข้อง
- จัดกรอบการตอบอย่างเหมาะสม

## ตัวอย่างใช้จริง

**ไม่มีบทบาท:**
```
เขียนสรุปบทความนี้
```

**มีบทบาท:**
```
คุณเป็นผู้ช่วยฝ่ายบริหารที่ยุ่ง สรุปบทความนี้ให้ฉันอ่านใน 30 วินาที เน้นเฉพาะสิ่งที่นำไปปฏิบัติได้จริง
```

บทบาทบอก AI ให้โฟกัสสิ่งที่เป็นประโยชน์สำหรับคนยุ่ง

## บทบาทยอดนิยม

- "ทำหน้าที่เป็น [อาชีพ]"
- "คุณเป็น [บทบาท] ที่มีประสบการณ์ [จำนวน] ปี"
- "สมมติว่าคุณเป็น [บุคลิก]"
- "ในฐานะผู้เชี่ยวชาญด้าน [สาขา]"

ทดลอง! บทบาทต่างกันให้ผลลัพธ์ต่างกันมาก
