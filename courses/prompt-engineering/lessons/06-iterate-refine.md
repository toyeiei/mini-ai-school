# Iterate & Refine

The first response is rarely the best. Refine until it's right.

## The Iterate Loop

```
Ask → Review → Refine → Review → Refine → Done
```

Think of it like editing a document. First draft → revisions → final version.

## Types of Refinements

**Add more:**
```
Good start! Now add a section about pricing.
```

**Make it simpler:**
```
This is too technical. Simplify it for someone with no background in finance.
```

**Change tone:**
```
Great! Now rewrite it with a more playful, humorous tone.
```

**Be more specific:**
```
The examples are helpful. Can you add more specific numbers and statistics?
```

## Ask for Alternatives

```
Give me 3 different versions of this headline:
1. Serious and professional
2. Playful and catchy
3. Focused on the benefit
```

Then pick elements you like from each.

## The "Make it better" Pattern

```
Here is [content]. Make it better by:
- Making the sentences shorter
- Adding more action verbs
- Removing any repetitive phrases
```

## When to Stop

- When it answers what you need
- When you've gone through 3-4 iterations
- When further changes feel like "polishing a turd" (start over instead)

## Key Mindset

Good prompting is a conversation, not a one-shot. Expect to refine. This is normal and efficient.
