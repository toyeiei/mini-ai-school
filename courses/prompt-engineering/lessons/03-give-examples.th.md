# ให้ตัวอย่าง

ตัวอย่างหนึ่งมีค่ามากกว่าพันคำ

## ทำไมตัวอย่างถึงได้ผล

เมื่อคุณให้ตัวอย่าง คุณแสดง AI ว่าคุณต้องการอะไร—ไม่ใช่แค่อธิบาย

## ตัวอย่าง: สไตล์การเขียน

**ไม่มีตัวอย่าง:**
```
Write product descriptions for our online store.
```

**มีตัวอย่าง:**
```
Write product descriptions for our online store. Here's an example of the style we want:

[Example]
Introducing the UltraSoft Towel Set. Made from 100% organic cotton, these towels combine luxury with everyday durability. The set includes 2 bath towels, 2 hand towels, and 4 washcloths. Machine washable for easy care.
[/Example]
```

AI เข้าใจเสียงแบรนด์ของคุณจากตัวอย่าง

## ตัวอย่าง: รูปแบบ

**ไม่มีตัวอย่าง:**
```
Summarize this article.
```

**มีตัวอย่าง:**
```
Summarize this article in 3 bullet points. Here's the format I want:

- **Main point:** [one sentence]
- **Key detail:** [one sentence]  
- **Why it matters:** [one sentence]
```

## ประเภทของตัวอย่าง

1. **ตัวอย่างสไตล์:** "เขียนแบบนี้..."
2. **ตัวอย่างรูปแบบ:** "จัดโครงสร้างแบบนี้..."
3. **ตัวอย่างคุณภาพ:** "ระดับความละเอียดแบบนี้..."
4. **ตัวอย่างเนื้อหา:** "รวมสิ่งเช่น..."

## เคล็ดลับขั้นสูง: รวมสิ่งที่ไม่ควรทำ

```
Write like this: "The sunset painted the sky in brilliant orange."

NOT like this: "The sunset was very beautiful and colorful."
```

การแสดงตัวอย่างที่ไม่ดีช่วย AI หลีกเลี่ยงมัน
