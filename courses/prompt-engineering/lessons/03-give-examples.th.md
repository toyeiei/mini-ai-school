# ให้ตัวอย่าง

ตัวอย่างหนึ่งมีค่ามากกว่าพันคำ

## ทำไมตัวอย่างถึงได้ผล

เมื่อคุณให้ตัวอย่าง คุณแสดง AI ว่าคุณต้องการอะไร—ไม่ใช่แค่อธิบาย

## ตัวอย่าง: สไตล์การเขียน

**ไม่มีตัวอย่าง:**
```
เขียนคำอธิบายสินค้าสำหรับร้านออนไลน์ของเรา
```

**มีตัวอย่าง:**
```
เขียนคำอธิบายสินค้าสำหรับร้านออนไลน์ของเรา นี่คือตัวอย่างสไตล์ที่เราต้องการ:

[ตัวอย่าง]
แนะนำชุดผ้าขนหนู UltraSoft ทำจากผ้าฝ้ายออร์แกนิก 100% ผ้าขนหนูเหล่านี้ผสมผสานความหรูหรากับความทนทานใช้งานทุกวัน ชุดประกอบด้วยผ้าขนหนูอาบน้ำ 2 ผืน ผ้าเช็ดมือ 2 ผืน และผ้าขนนุ่ม 4 ผืน ซักด้วยเครื่องได้สำหรับการดูแลง่าย
[/ตัวอย่าง]
```

AI เข้าใจเสียงแบรนด์ของคุณจากตัวอย่าง

## ตัวอย่าง: รูปแบบ

**ไม่มีตัวอย่าง:**
```
สรุปบทความนี้
```

**มีตัวอย่าง:**
```
สรุปบทความนี้เป็น 3 ข้อ นี่คือรูปแบบที่ฉันต้องการ:

- **จุดหลัก:** [ประโยคเดียว]
- **รายละเอียดสำคัญ:** [ประโยคเดียว]
- **ทำไมสำคัญ:** [ประโยคเดียว]
```

## ประเภทของตัวอย่าง

1. **ตัวอย่างสไตล์:** "เขียนแบบนี้..."
2. **ตัวอย่างรูปแบบ:** "จัดโครงสร้างแบบนี้..."
3. **ตัวอย่างคุณภาพ:** "ระดับความละเอียดแบบนี้..."
4. **ตัวอย่างเนื้อหา:** "รวมสิ่งเช่น..."

## เคล็ดลับขั้นสูง: รวมสิ่งที่ไม่ควรทำ

```
เขียนแบบนี้: "พระอาทิตย์ตกทาสีท้องฟ้าเป็นสีส้มสุกสว่าง"

ไม่ใช่แบบนี้: "พระอาทิตย์ตกสวยมากและมีสีหลากหลาย"
```

การแสดงตัวอย่างที่ไม่ดีช่วย AI หลีกเลี่ยงมัน
