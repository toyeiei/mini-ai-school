# ให้ชัดเจนและเฉพาะเจาะจง

คุณยิ่งเฉพาะเจาะจงมากเท่าไหร่ ผลลัพธ์ยิ่งดีขึ้นเท่านั้น

## แทนที่จะเป็น "เขียนอะไรสักอย่าง"

บอกมันว่า:
- เขียนอะไร
- สำหรับใคร
- ยาวแค่ไหน
- โทนอะไร

## ตัวอย่าง: ขอเขียนอีเมล

**คลุมเครือ:**
```
เขียนอีเมลถึงหัวหน้า
```

**เฉพาะเจาะจง:**
```
เขียนอีเมลทางการถึงผู้จัดการอธิบายว่าโปรเจกต์จะล่าช้ากว่ากำหนด 3 วันเนื่องจากปัญหาทางเทคนิคที่ไม่คาดคิด เขียนไม่เกิน 150 คำ ขอโทษแต่มั่นใจว่าคุณภาพจะไม่ได้รับผลกระทบ
```

เวอร์ชันเฉพาะเจาะจงให้ AI ทุกอย่างที่ต้องการ

## เคล็ดลับความเฉพาะเจาะจง

1. **กำหนดกลุ่มเป้าหมาย:** "สำหรับเด็ก 10 ขวบ" vs "สำหรับวิศวกรซอฟต์แวร์"
2. **กำหนดความยาว:** "3 ประโยค" vs "รายงานแบบละเอียด"
3. **เลือกโทน:** "เป็นกันเองและเป็นมิตร" vs "เป็นทางการและมืออาชีพ"
4. **บอกวัตถุประสงค์:** "เพื่อขอโทษ" vs "เพื่อโน้มน้าวให้เปลี่ยนใจ"

## ฝึกฝน

นำ prompt คลุมเครือนี้มาทำให้เฉพาะเจาะจง:

```
เขียนเรื่องเกี่ยวกับภาวะโลกร้อน
```

**เวอร์ชันของคุณ:** ___________________________________

**ตัวอย่างปรับปรุง:**
```
เขียนภาพรวม 300 คำเกี่ยวกับภาวะโลกร้อนสำหรับนักเรียนมัธยม อธิบายวิทยาศาสตร์พื้นฐานอย่างง่าย รวมตัวอย่างว่ามันส่งผลต่อชีวิตประจำวันอย่างไร โทน: ให้ความรู้แต่ไม่ทำให้ตกใจ
```

## เมื่อไม่แน่ใจ ให้อธิบายมากกว่า

AI สามารถเพิกเฉยรายละเอียดที่เกินไปได้ง่ายกว่าการเดาว่าคุณหมายถึงอะไร อย่ากลัวที่จะอธิบายอย่างละเอียด
