# แบ่งเป็นขั้นตอน

งานซับซ้อนต้องแบ่งเป็นขั้นตอนง่ายๆ

## ทำไมต้องแบ่ง?

Prompt เดียวขอแผนธุรกิจสมบูรณ์มักให้ผลลัพธ์ทั่วไปและผิวเผิน

แต่ขอแต่ละส่วนแยกกันให้คุณได้เนื้อหาที่ละเอียดและคิดอย่างมีสาระ

## วิธีขั้นตอน

แทนที่จะเป็น:
```
เขียนแผนธุรกิจสมบูรณ์
```

ลอง:
```
1. ก่อนอื่น ช่วยฉันระบุลูกค้าเป้าหมาย ใครน่าจะซื้อสินค้านี้มากที่สุด?
2. จากนั้น ปัญหาหลัก 3 อย่างที่สินค้าของฉันแก้ให้พวกเขาคืออะไร?
3. ต่อไป ฉันควรตั้งราคาเท่าไหร่? พิจารณาสินค้าที่คล้ายกันในตลาด
4. สุดท้าย เขียนย่อหนึ่งย่อหน้ารวมทุกอย่างข้างบน
```

## เมื่อไหร่ควรใช้

- **เอกสารวางแผน:** ทำทีละส่วน
- **โค้ด:** ได้โครงสร้างก่อน แล้วค่อยทำแต่ละส่วน
- **วิจัย:** รวบรวมข้อมูลเป็นขั้นตอนก่อนสังเคราะห์
- **งานสร้างสรรค์:** ร่างเค้าโครงก่อน แล้วขยายแต่ละส่วน

## ตัวอย่างจริง

**Prompt ใหญ่หนึ่งอัน:**
```
เขียนเว็บไซต์สมบูรณ์สำหรับร้านเบเกอรี่ของฉัน
```

**แบ่งเป็นขั้นตอน:**
```
1. เว็บไซต์ร้านเบเกอรี่โดยทั่วไปต้องมีหน้าอะไรบ้าง?
2. เขียนข้อความหลักหน้าแรกสำหรับร้านเบเกอรี่ครอบครัวเน้นของบ้านและวัตถุดิบท้องถิ่น
3. สร้างโครงสร้างเมนู 5 หมวดหมู่
4. เขียนย่อหน้า "เกี่ยวกับเรา" สำหรับร้านเบเกอรี่ครอบครัวที่เปิดตั้งแต่ปี 1995
```
