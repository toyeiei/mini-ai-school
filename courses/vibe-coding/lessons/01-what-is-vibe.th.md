# What is Vibe Coding

Vibe coding คือแนวทางใหม่ในการพัฒนาซอฟต์แวร์ที่ใช้ AI assistant ช่วยเร่งกระบวนการสร้างสรรค์ แทนที่จะเขียนโค้ดทุกบรรทัดเอง คุณจะเป็นผู้นำ AI ในการสร้างสิ่งที่คุณต้องการ

## The Core Idea

Traditional coding: คุณเขียนโค้ดทีละบรรทัด

Vibe coding: คุณอธิบายสิ่งที่ต้องการ แล้ว AI ช่วยสร้างมันให้ คุณทำหน้าที่ตรวจสอบ ปรับปรุง และกำกับทิศทาง

## What Makes It Different

| Traditional | Vibe Coding |
|--------------|-------------|
| เขียนทุกบรรทัดเอง | สร้างโค้ดด้วย AI |
| ค้นหาเอกสารเอง | ถาม AI เพื่อดูบริบท |
| Debug คนเดียว | ทำงานร่วมกับ AI |
| เรียนรู้ syntax ก่อน | เรียนรู้ไปพร้อมกับสร้าง |
| รอบพัฒนายาวนาน | Prototype ได้รวดเร็ว |

## The Vibe Coding Workflow

1. **Describe** - บอก AI ว่าคุณอยากสร้างอะไร
2. **Generate** - AI สร้างโค้ดให้
3. **Review** - คุณตรวจสอบผลลัพธ์
4. **Refine** - ขอให้แก้ไขหรือปรับปรุง
5. **Repeat** - ทำซ้ำจนกว่าจะเสร็จ

## What You Still Need

Vibe coding ไม่ได้ลดความสำคัญของความรู้ด้าน programming คุณยังคงต้องมี:

- **Problem solving** - เข้าใจว่าจะสร้างอะไร
- **Critical thinking** - ประเมินข้อเสนอแนะของ AI
- **Debugging** - หาและแก้ไขปัญหา
- **Architecture** - จัดโครงสร้างแอปพลิเคชันขนาดใหญ่
- **Testing** - ตรวจสอบความถูกต้อง

## The Human-AI Partnership

มอง AI เหมือนนักพัฒนารุ่นจูเนียร์ที่ทำงานเร็วมาก:
- ไม่เคยเหนื่อย
- รู้ pattern หลายแบบ
- เขียนโค้ดเร็ว
- ทำผิดพลาดได้
- ต้องการคำแนะนำที่ชัดเจน

คุณเป็นผู้ให้:
- Vision และทิศทาง
- การควบคุมคุณภาพ
- บริบทเกี่ยวกับผู้ใช้
- การตัดสินใจสุดท้าย

## Why Learn Vibe Coding?

วงการพัฒนาซอฟต์แวร์กำลังเปลี่ยนแปลง นักพัฒนาที่ประสบความสำเร็จคือผู้ที่สามารถทำงานร่วมกับ AI ได้อย่างมีประสิทธิภาพ ไม่ใช่แค่เขียนโค้ดเอง คอร์สนี้สอนคุณวิธีใช้ AI เป็นพาร์ทเนอร์ในการสร้างซอฟต์แวร์
