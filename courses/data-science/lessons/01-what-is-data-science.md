# What is Data Science

Data science is the practice of extracting knowledge from data. It combines statistics, programming, and domain expertise to solve problems.

## The Data Science Process

1. **Ask a Question** - What do you want to know?
2. **Collect Data** - Gather the data you need
3. **Explore and Clean** - Understand and prepare the data
4. **Analyze** - Find patterns and insights
5. **Communicate** - Share your findings

## What Data Scientists Do

Data scientists work with data to:
- Predict outcomes (will this customer leave?)
- Classify things (is this email spam?)
- Detect anomalies (is this transaction fraud?)
- Optimize decisions (what price maximizes sales?)

## The Tools

### Programming Languages

**Python** is the most popular for data science:
- Simple syntax, easy to learn
- Powerful libraries for analysis
- Great for automation

```python
# Python example
import pandas as pd

# Load data
df = pd.read_csv('sales.csv')

# Calculate average
avg_sales = df['revenue'].mean()
print(f"Average sales: ${avg_sales}")
```

**R** is also common, especially in academia and statistics.

## A Simple Example

You have data on daily ice cream sales:

```python
sales = [120, 89, 156, 201, 178, 134, 167, 189, 145, 198]

# Average sales
average = sum(sales) / len(sales)
print(f"Average: {average}")  # Output: 157.7
```

That's data science at its simplest: calculating and interpreting numbers.
