# What is Data Science

Data science คือการนำความรู้ออกมาจากข้อมูล มันผสมผสานสถิติ การเขียนโปรแกรม และความเชี่ยวชาญด้านโดเมนเพื่อแก้ปัญหาต่าง ๆ

## กระบวนการทำ Data Science

1. **ตั้งคำถาม** - คุณอยากรู้อะไร?
2. **เก็บข้อมูล** - รวบรวมข้อมูลที่คุณต้องการ
3. **สำรวจและทำความสะอาด** - เข้าใจและเตรียมข้อมูล
4. **วิเคราะห์** - หา pattern และ insight
5. **สื่อสาร** - แชร์ผลการวิเคราะห์

## Data Scientist ทำอะไรบ้าง

Data scientist ทำงานกับข้อมูลเพื่อ:
- ทำนายผลลัพธ์ (ลูกค้าคนนี้จะออกไปหรือไม่?)
- จัดหมวดหมู่ (อีเมลนี้เป็น spam หรือไม่?)
- ตรวจจับสิ่งผิดปกติ (ธุรกรรมนี้เป็นการหลอกลวงหรือไม่?)
- ปรับปรุงการตัดสินใจ (ราคาเท่าไหร่จะขายดีที่สุด?)

## เครื่องมือต่าง ๆ

### Programming Languages

**Python** เป็นที่นิยมมากที่สุดสำหรับ data science:
- ไวยากรณ์ง่าย เรียนรู้สะดวก
- Library ที่ทรงพลังสำหรับการวิเคราะห์
- เหมาะสำหรับ automation

```python
# Python example
import pandas as pd

# Load data
df = pd.read_csv('sales.csv')

# Calculate average
avg_sales = df['revenue'].mean()
print(f"Average sales: ${avg_sales}")
```

**R** ก็ใช้กันมากเช่นกัน โดยเฉพาะในวงการวิชาการและสถิติ

### Library ที่สำคัญ

| Library | วัตถุประสงค์ |
|---------|---------|
| pandas | จัดการข้อมูล |
| numpy | การคำนวณเชิงตัวเลข |
| matplotlib | การวาดกราฟเบื้องต้น |
| scikit-learn | Machine learning |

## ทำไมต้องเรียน Data Science?

ทักษะ data science ช่วยให้คุณ:
- ตัดสินใจโดยอ้างอิงหลักฐาน
- เข้าใจโลกรอบตัวผ่านข้อมูล
- สร้างโมเดลทำนาย
- ทำให้การวิเคราะห์เป็นอัตโนมัติ

## ใครใช้ Data Science บ้าง?

- **ธุรกิจ** - พฤติกรรมลูกค้า, ทำนายยอดขาย
- **สุขภาพ** - ทำนายโรค, ค้นหายา
- **กีฬา** - ประสิทธิภาพนักกีฬา, กลยุทธ์
- **การเงิน** - ประเมินความเสี่ยง, ตรวจจับการหลอกลวง
- **วิทยาศาสตร์** - วิเคราะห์การวิจัย, การทดลอง

## คุณจะเรียนอะไรบ้าง

คอร์สนี้ครอบคลุมพื้นฐาน data science:
1. เข้าใจ data types ต่าง ๆ
2. การวัดทางสถิติ (mean, median ฯลฯ)
3. การ visualization ข้อมูล
4. พื้นฐานความน่าจะเป็น
5. การทดสอบสมมติฐาน
6. การวิเคราะห์ regression
7. บทนำ machine learning

ไม่จำเป็นต้องมีปริญญาด้านคณิตศาสตร์ เรามุ่งเน้นแนวคิดที่นำไปใช้ได้จริง

## ตัวอย่างง่าย ๆ

คุณมีข้อมูลยอดขายไอศครีมรายวัน:

```python
sales = [120, 89, 156, 201, 178, 134, 167, 189, 145, 198]

# Average sales
average = sum(sales) / len(sales)
print(f"Average: {average}")  # Output: 157.7
```

นั่นคือ data science ในรูปแบบที่ง่ายที่สุด: คำนวณและตีความตัวเลข

## Mindset ของ Data Scientist ที่ดี

Data scientist ที่ดี:
- ตั้งคำถามก่อนจะกระโดดไปหาคำตอบ
- ตั้งคำถามกับสิ่งที่เชื่อ
- แสดงความอยากรู้เกี่ยวกับ pattern ที่ไม่คาดคิด
- สื่อสารให้เข้าใจง่าย

มาเริ่มสำรวจข้อมูลกันเลย
