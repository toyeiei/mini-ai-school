# บทนำสู่การเขียนโปรแกรม

ยินดีต้อนรับสู่ Code School ในบทเรียนนี้ คุณจะได้เรียนรู้ว่าการเขียนโปรแกรมคืออะไร และทำไมมันถึงสำคัญ

## การเขียนโปรแกรมคืออะไร?

การเขียนโปรแกรมคือศาสตร์แห่งการให้คำสั่งแก่คอมพิวเตอร์ เหมือนกับการทำตามสูตรอบขนมปัง คอมพิวเตอร์ก็ทำตามโปรแกรมเพื่อปฏิบัติงานต่างๆ

## ทำไมต้องเรียนเขียนโปรแกรม?

- **การแก้ปัญหา**: การเขียนโปรแกรมสอนคุณให้แบ่งปัญหาที่ซับซ้อนออกเป็นขั้นตอนเล็กๆ ที่จัดการได้ง่าย
- **ความคิดสร้างสรรค์**: สร้างเว็บไซต์ แอป เกม และเครื่องมือที่แก้ปัญหาในชีวิตจริง
- **โอกาสอาชีพ**: ทักษะด้านเทคโนโลยีมีความต้องการสูงในทุกอุตสาหกรรม

## คุณสามารถสร้างอะไรได้บ้าง?

- เว็บแอปพลิเคชันและเว็บไซต์
- แอปมือถือสำหรับ iOS และ Android
- ซอฟต์แวร์บนเดสก์ท็อป
- เกมและประสบการณ์แบบอินเทอแอคทีฟ
- สคริปต์อัตโนมัติ
- โมเดล Machine learning

## เริ่มต้นอย่างไร

ในคอร์สนี้ เราจะใช้ JavaScript เป็นภาษาหลัก เพราะเป็นภาษาที่เหมาะกับผู้เริ่มต้น ใช้กันแพร่หลาย และทำงานได้ในทุกเบราว์เซอร์

> "ทุกคนในประเทศนี้ควรเรียนรู้การเขียนโปรแกรม เพราะมันสอนคุณให้คิด" - Steve Jobs

## แนวคิดแรกของคุณ

จุดสำคัญของการเขียนโปรแกรมคือคอมพิวเตอร์ไม่เข้าใจนัยย่อย พวกมันทำตามคำสั่งตรงๆ ตามที่เขียนไว้ ความละเอียดถูกต้องนี้ทั้งเป็นทั้งความท้าทายและพลังพิเศษ

มาเริ่มต้นการเดินทางสู่โลกแห่งการเขียนโปรแกรมกันเลย!
