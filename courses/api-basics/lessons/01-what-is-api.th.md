# API คืออะไร

API (Application Programming Interface) ช่วยให้แอปสามารถสื่อสารกันได้

## ตัวอย่างจากชีวิตจริง

เปรียบเสมือนร้านอาหาร:
- **คุณ** (client) สั่งอาหาร
- **เมนู** (API) แสดงรายการที่มีอยู่
- **ครัว** (server) จัดเตรียมอาหารให้คุณ
- คุณไม่จำเป็นต้องเข้าไปในครัว - คุณใช้เมนู

API ทำงานในลักษณะเดียวกัน:
- แอปส่งคำขอข้อมูลผ่าน API
- Server ประมวลผลและส่งข้อมูลกลับ
- Client ไม่เห็นการทำงานภายใน

## ประเภทของ API

| ประเภท | ตัวอย่าง |
|--------|---------|
| **Web APIs** | REST, GraphQL |
| **Library APIs** | jQuery, React |
| **Operating System** | Windows API, macOS API |

เราจะเน้นไปที่ **Web APIs**

## โมเดล Client-Server

```
┌─────────┐     Request      ┌─────────┐
│  Client │ ───────────────→ │ Server  │
│ (You)   │ ←─────────────── │ (API)   │
└─────────┘     Response     └─────────┘
```

## ทำไม API จึงสำคัญ

- แอปแชร์ข้อมูลกัน (อากาศ, แผนที่, การชำระเงิน)
- แอปมือถือเชื่อมต่อกับ backend
- สถาปัตยกรรม Microservices
- การรวมเข้ากับบริการของบุคคลที่สาม

## สรุป

- API = เมนูสำหรับโปรแกรม
- Clients ส่งคำขอ, servers ตอบกลับ
- ทำให้แอปสามารถสื่อสารกันได้
- Web APIs มีอยู่ทุกที่
