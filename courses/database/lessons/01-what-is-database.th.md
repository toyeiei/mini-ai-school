# Database คืออะไร

Database คือชุดข้อมูลที่จัดระเบียบและเก็บไว้ในรูปแบบดิจิทัล ช่วยให้คุณจัดเก็บ ค้นหา และจัดการข้อมูลได้อย่างมีประสิทธิภาพ

## ทำไม Database ถึงสำคัญ

ลองนึกภาพแอปทุกแอปที่คุณใช้อยู่:
- **Social media** เก็บโพสต์ เพื่อน และข้อความของคุณ
- **ร้านค้าออนไลน์** ติดตามสินค้า ออเดอร์ และลูกค้า
- **ธนาคาร** จัดการยอดเงินในบัญชีและธุรกรรม

ข้อมูลทั้งหมดนี้ถูกเก็บอยู่ใน database

## ประเภทของ Database

### Relational Database (SQL)

ข้อมูลถูกจัดเป็นตารางที่มีแถวและคอลัมน์ ตารางต่างๆ สามารถเชื่อมโยงกันได้

ตัวอย่าง: MySQL, PostgreSQL, SQLite

```
-- A table called "users"
+----+------------------+---------+
| id | email            | name    |
+----+------------------+---------+
| 1  | alice@email.com  | Alice   |
| 2  | bob@email.com    | Bob     |
+----+------------------+---------+
```

### NoSQL Database

ข้อมูลถูกเก็บในรูปแบบยืดหยุ่น เช่น JSON documents, key-value pairs, หรือ graphs

ตัวอย่าง: MongoDB, Redis, Cassandra

```json
{ "users": [
    { "id": 1, "email": "alice@email.com", "name": "Alice" },
    { "id": 2, "email": "bob@email.com", "name": "Bob" }
] }
```

## Database Management System (DBMS)

DBMS คือซอฟต์แวร์ที่ให้คุณโต้ตอบกับ database โดยจัดการ:
- การเก็บข้อมูล
- การค้นหาข้อมูล
- การรักษาความปลอดภัยข้อมูล
- การจัดการการเข้าถึงพร้อมกัน

## SQL vs NoSQL

| ด้าน | SQL | NoSQL |
|--------|-----|-------|
| โครงสร้าง | ตารางที่固定 | ยืดหยุ่น |
| การค้นหา | ภาษามาตรฐาน | แตกต่างกันไปตามประเภท |
| การขยายตัว | แนวตั้ง (Vertical) | แนวนอน (Horizontal) |
| เหมาะกับ | ข้อมูลที่เกี่ยวข้องกัน | การพัฒนาที่รวดเร็ว |

## สิ่งที่คุณจะเรียนรู้

คอร์สนี้ครอบคลุม relational database โดยใช้ SQL เพราะ:
- SQL ใช้กันอย่างแพร่หลายและเป็นมาตรฐาน
- แนวคิดสามารถนำไปใช้กับ database ส่วนใหญ่ได้
- เหมาะสำหรับการเรียนรู้เกี่ยวกับความสัมพันธ์ของข้อมูล

เมื่อจบคอร์ส คุณจะเข้าใจการออกแบบ สร้าง และค้นหา database

## คำศัพท์สำคัญ

- **Table** - ชุดข้อมูลที่เกี่ยวข้องกัน (เหมือนสเปรดชีต)
- **Row** - ข้อมูลหนึ่งรายการในตาราง
- **Column** - ฟิลด์ข้อมูลเฉพาะด้าน
- **Primary Key** - ตัวระบุเฉพาะสำหรับแต่ละแถว
- **Foreign Key** - ลิงก์ไปยังตารางอื่น
