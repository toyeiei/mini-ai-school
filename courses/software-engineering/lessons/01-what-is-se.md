# What is Software Engineering

Software engineering is the practice of designing, building, and maintaining software systems. It combines computer science, engineering principles, and project management to create reliable, scalable, and maintainable software.

## The Software Development Life Cycle

Every software project follows a cycle:

1. **Requirements** - What should the software do?
2. **Design** - How will we build it?
3. **Implementation** - Writing the code
4. **Testing** - Verifying it works
5. **Deployment** - Releasing to users
6. **Maintenance** - Fixing issues and adding features

## Key Differences from Coding

Programming focuses on writing code. Software engineering encompasses the entire lifecycle:

- **Coding** is writing instructions for computers
- **Software Engineering** is applying systematic practices to create software that solves real problems

## Core Skills of a Software Engineer

- **Problem decomposition** - Breaking complex problems into smaller pieces
- **Communication** - Understanding requirements and explaining solutions
- **Debugging** - Finding and fixing issues systematically
- **Testing** - Ensuring code works correctly
- **Version control** - Managing code changes over time
- **Reading documentation** - Understanding how tools and frameworks work
