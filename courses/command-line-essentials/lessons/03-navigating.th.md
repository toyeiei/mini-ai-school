# นำทางไฟล์

Terminal มีระบบไฟล์เหมือน Finder หรือ File Explorer มาเรียนวิธีเคลื่อนที่ไปมากัน

## ฉันอยู่ที่ไหน? `pwd`

`pwd` ย่อมาจาก "print working directory" มันแสดงว่าคุณอยู่ที่ไหนอย่างแม่นยำ:

```bash
pwd
```

ผลลัพธ์:
```
/Users/kasidis/Projects
```

นึกภาพเหมือนถาม Google Maps: "ตอนนี้ฉันอยู่ที่ไหน?"

## มีอะไรบ้าง? `ls`

`ls` แสดงรายการไฟล์และโฟลเดอร์ในตำแหน่งปัจจุบัน:

```bash
ls
```

ผลลัพธ์:
```
Desktop    Downloads    Documents    Pictures
```

**ใช้จริง:** ดูว่ามีไฟล์อะไรอยู่ก่อนเปิด

## ไปที่ไหน `cd`

`cd` หมายถึง "change directory" คลิกสองครั้งที่โฟลเดอร์:

```bash
cd Documents
```

ตอนนี้คุณอยู่ในโฟลเดอร์ Documents ลองพิมพ์ `pwd` อีกครั้งเพื่อยืนยัน

## กลับไป `cd ..`

สองจุด `..` หมายถึง "โฟลเดอร์แม่" (โฟลเดอร์ที่ครอบโฟลเดอร์ปัจจุบัน)

```bash
cd ..
```

นี่จะพาคุณกลับไปโฟลเดอร์ก่อนหน้า

## กลับบ้าน `cd ~` หรือแค่ `cd`

เครื่องหมาย `~` หมายถึงโฟลเดอร์หลักของคุณเสมอ:

```bash
cd ~
```

หรือแค่พิมพ์ `cd` โดยไม่ต้องใส่อะไร

## เคล็ดลับ: เติมคำอัตโนมัติ

กด `Tab` ขณะพิมพ์เพื่อเติมชื่อโฟลเดอร์อัตโนมัติ:

```bash
cd Docu[Tab]
```

Terminal เติม "Documents" ให้คุณ สะดวกใช่ไหม?

## สรุปเร็ว

| คำสั่ง | ทำอะไร |
|--------|--------|
| `pwd` | ฉันอยู่ที่ไหน? |
| `ls` | มีอะไรบ้าง? |
| `cd โฟลเดอร์` | เข้าไปในโฟลเดอร์ |
| `cd ..` | กลับไป |
| `cd ~` | กลับบ้าน |

ฝึก: `cd` ไป Desktop พิมพ์ `ls` แล้ว `cd ..` กลับมา
