# คัดลอกและย้ายไฟล์

เรียนวิธีคัดลอกและย้ายไฟล์—เหมือนลากแล้ววาง แต่ด้วยคำสั่ง

## คัดลอกไฟล์ `cp`

`cp` หมายถึง "copy":

```bash
cp readme.txt readme-backup.txt
```

ตอนนี้คุณมีสองไฟล์: ต้นฉบับและสำรอง

**ใช้จริง:** ทำสำรองก่อนแก้ไขไฟล์สำคัญ

## คัดลอกโฟลเดอร์ `cp -r`

สำหรับโฟลเดอร์ เพิ่ม `-r` (recursive):

```bash
cp -r my-project my-project-backup
```

นี่คัดลอกทุกอย่างข้างในด้วย

## ย้าย/เปลี่ยนชื่อไฟล์ `mv`

`mv` หมายถึง "move" คุณใช้มันเปลี่ยนชื่อได้ด้วย:

```bash
mv readme.txt documents/
```

นี่ย้าย "readme.txt" ไปในโฟลเดอร์ "documents"

เปลี่ยนชื่อ:
```bash
mv old-name.txt new-name.txt
```

**ใช้จริง:** จัดไฟล์เข้าโฟลเดอร์ หรือแก้ตกสะกดในชื่อไฟล์

## สรุปเร็ว

| คำสั่ง | ทำอะไร |
|--------|--------|
| `cp ไฟล์ สำเนา` | คัดลอกไฟล์ |
| `cp -r โฟลเดอร์ สำเนา` | คัดลอกโฟลเดอร์ |
| `mv ไฟล์ โฟลเดอร์` | ย้ายไฟล์ |
| `mv เก่า ใหม่` | เปลี่ยนชื่อ |

## แพทเทิร์นที่เป็นประโยชน์

คัดลอกพร้อมวันที่สำหรับเวอร์ชัน:

```bash
cp readme.txt readme-2025-04-03.txt
```

แบบนี้คุณรู้ว่าแต่ละสำรองถูกสร้างเมื่อไหร่
