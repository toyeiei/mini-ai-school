# คำสั่งแรกของคุณ

ถึงเวลาพิมพ์คำสั่งแรก มาเริ่มจากสามคำสั่งง่ายๆ

## คำสั่ง `echo`

`echo` พูดซ้ำสิ่งที่คุณบอก เหมือนนกแก้ว:

```bash
echo "สวัสดี ชาวโลก!"
```

ผลลัพธ์:
```
สวัสดี ชาวโลก!
```

**ใช้จริง:** ทดสอบว่า Terminal ทำงานได้ไหม หรือพิมพ์ข้อความในสคริปต์

## คำสั่ง `date`

`date` แสดงวันที่และเวลาปัจจุบัน:

```bash
date
```

ผลลัพธ์:
```
พฤ. เม.ย.  3 12:30:00 +07 2025
```

**ใช้จริง:** ตรวจสอบเวลา เพิ่มวันที่ให้ไฟล์สำรอง

## คำสั่ง `whoami`

`whoami` ถามว่า "ฉันล็อกอินเป็นใคร?"

```bash
whoami
```

ผลลัพธ์:
```
kasidis
```

**ใช้จริง:** ยืนยันว่าคุณเป็นผู้ใช้ที่ถูกต้องก่อนทำสิ่งสำคัญ

## ลองทำ

พิมพ์คำสั่งเหล่านี้ทีละคำสั่ง:

1. `echo "ฉันชื่อ..."`
2. `date`
3. `whoami`

สังเกตไหมว่าแต่ละคำสั่งทำเสร็จเร็ว? Terminal เร็วเพราะไม่ต้องโหลดปุ่มหรือกราฟิก มันแค่ทำสิ่งที่คุณขอเท่านั้น

## ถ้าพิมพ์ผิดล่ะ?

ถ้าคุณพิมพ์ผิด คุณจะเห็นข้อผิดพลาด:

```bash
ech0 "อ๊าบ"
```

ผลลัพธ์:
```
-bash: ech0: command not found
```

คอมพิวเตอร์บอกเป็นภาษาอังกฤษว่า: "ฉันไม่รู้จักคำสั่งนั้น" นี่เป็นเรื่องปกติตอนเรียน แค่พิมพ์ใหม่ด้วยคำสะกดที่ถูกต้อง
