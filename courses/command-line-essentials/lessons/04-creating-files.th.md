# สร้างและลบไฟล์

ตอนนี้คุณรู้วิธีมองรอบๆ แล้ว มาเรียนสร้างและลบกัน

## สร้างโฟลเดอร์ `mkdir`

`mkdir` หมายถึง "make directory":

```bash
mkdir my-project
```

ตอนนี้โฟลเดอร์ชื่อ "my-project" ถูกสร้างแล้ว ยืนยันได้ด้วย `ls`

**ใช้จริง:** สร้างโฟลเดอร์โปรเจกต์ก่อนเริ่มเขียนโค้ด

## สร้างไฟล์ `touch`

`touch` สร้างไฟล์ว่าง:

```bash
touch readme.txt
```

นี่คือสร้างไฟล์ว่างชื่อ "readme.txt"

**ใช้จริง:** สร้างไฟล์ที่ว่างเปล่า ไฟล์บันทึก หรือไฟล์ที่คุณต้องการ

## ลบไฟล์ `rm`

`rm` หมายถึง "remove" ใช้อย่างระวัง—ไม่มีถังขยะ!

```bash
rm readme.txt
```

ไฟล์หายไปตลอดกาล

## ลบโฟลเดอร์ `rm -r`

สำหรับโฟลเดอร์ เพิ่ม `-r` (recursive):

```bash
rm -r my-project
```

นี่จะลบโฟลเดอร์และทุกอย่างข้างใน

## ระวัง!

ไม่มี "ย้อนกลับ" ใน Terminal ลบแล้วหายเลย

วิธีปลอดภัย: พิมพ์ `ls` ก่อนลบเสมอ เพื่อดูว่าจะลบอะไร

## เคล็ดลับความปลอดภัย

ใช้ `rm -i` เพื่อขอยืนยันก่อนลบ:

```bash
rm -i readme.txt
```

มันจะถาม: `remove readme.txt?` พิมพ์ `y` หมายถึงใช่ `n` หมายถึงไม่

## สรุปเร็ว

| คำสั่ง | ทำอะไร |
|--------|--------|
| `mkdir ชื่อ` | สร้างโฟลเดอร์ |
| `touch ชื่อ` | สร้างไฟล์ |
| `rm ชื่อ` | ลบไฟล์ |
| `rm -r ชื่อ` | ลบโฟลเดอร์ |
| `rm -i ชื่อ` | ลบพร้อมยืนยัน |
