# Terminal คืออะไร?

ลองนึกภาพว่าคุณมีผู้ช่วยส่วนตัว คุณอาจบอกเขาด้วยคำพูด หรือพิมพ์คำสั่งที่ชัดเจน Terminal ก็เหมือนการพิมพ์คำสั่งไปยังคอมพิวเตอร์ของคุณโดยตรง

แทนที่จะคลิกปุ่ม คุณพิมพ์คำสั่ง

## ทำไมต้องเรียน?

บทเรียนส่วนใหญ่บนอินเทอร์เน็ตต้องการให้คุณรู้จัก Terminal การเรียนรู้มันเปิดประตูไปสู่:
- การติดตั้งเครื่องมือสำหรับนักพัฒนา
- การใช้ AI coding assistants
- การทำงานกับเซิร์ฟเวอร์
- การทำงานอัตโนมัติ

## วิธีเปิด Terminal

**Mac:** กด `Cmd + Space` พิมพ์ "Terminal" แล้วกด Enter

**Windows:** กด `Win + X` เลือก "Terminal" หรือ "Command Prompt"

**Linux:** กด `Ctrl + Alt + T`

## มอง Terminal ครั้งแรก

คุณจะเห็นแบบนี้:

```
kasidis-MacBook:~ kasidis$
```

เรียกว่า **prompt** มันบอกคุณว่า:
- ชื่อคอมพิวเตอร์ (`kasidis-MacBook`)
- คุณอยู่ที่ไหน (`~` หมายถึงโฟลเดอร์หลัก)
- ชื่อผู้ใช้ (`kasidis`)

เคอร์เซอร์กระพริบ `|` กำลังรอคำสั่งของคุณ

## แนวคิดหลัก

นึกภาพ Terminal เป็นการสนทนากับคอมพิวเตอร์ คุณถาม มันตอบ ไม่ต้องคลิก ไม่ต้องลาก—แค่ข้อความ

> ในคอร์สนี้ เราจะพูด "ภาษาคอมพิวเตอร์" ด้วยกัน สุดท้ายแล้ว คุณจะรู้สึกเหมือนอยู่บ้านในโลกแบบข้อความนี้
