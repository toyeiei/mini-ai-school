# ค้นหาไฟล์

หาไฟล์และเนื้อหาโดยไม่ต้องคลิกผ่านโฟลเดอร์

## หาไฟล์ตามชื่อ `find`

`find` ค้นหาไฟล์:

```bash
find . -name "readme.txt"
```

นี่หาในโฟลเดอร์ปัจจุบัน (`.`) หาไฟล์ชื่อ "readme.txt"

**ใช้จริง:** หาไฟล์ที่ลืมว่าบันทึกไว้ที่ไหน

## หาไฟล์ตามประเภท `find`

```bash
find . -name "*.txt"
```

เครื่องหมาย `*` หมายถึง "อะไรก็ได้" นี่หาไฟล์ `.txt` ทั้งหมด

## ค้นหาในไฟล์ `grep`

`grep` ค้นหาข้อความในไฟล์:

```bash
grep "hello" readme.txt
```

นี่หาบรรทัดที่มี "hello" ใน readme.txt

**ใช้จริง:** หาว่ากำหนดตัวแปรไว้ที่ไหนในโค้ด หรือหาการตั้งค่าเฉพาะ

## ค้นหาแบบไม่สนตัวพิมพ์ `grep -i`

```bash
grep -i "hello" readme.txt
```

`-i` ทำให้ไม่สนตัวพิมพ์ หาทั้ง "Hello", "HELLO", "hello" ได้

## สรุปเร็ว

| คำสั่ง | ทำอะไร |
|--------|--------|
| `find . -name "ไฟล์"` | หาไฟล์ตามชื่อ |
| `find . -name "*.นามสกุล"` | หาไฟล์ตามนามสกุล |
| `grep "ข้อความ" ไฟล์` | หาข้อความในไฟล์ |
| `grep -i "ข้อความ" ไฟล์` | ค้นหาแบบไม่สนตัวพิมพ์ |

## ตัวอย่างใช้จริง

สมมติคุณมี 100 ไฟล์ และลืมว่าไฟล์ไหนเขียนว่า "TODO":

```bash
grep -r "TODO" .
```

`-r` หมายถึง recursive—มันค้นหาในทุกโฟลเดอร์ย่อยด้วย
