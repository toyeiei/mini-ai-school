# Next Steps

Congratulations! You've learned the Terminal essentials.

## What You Now Know

- Navigate folders (`pwd`, `ls`, `cd`)
- Create and delete files (`touch`, `mkdir`, `rm`)
- View file content (`cat`, `head`, `tail`, `less`)
- Copy and move files (`cp`, `mv`)
- Search for files and content (`find`, `grep`)
- Use keyboard shortcuts (`Ctrl + C`, `Tab`, etc.)
- Combine commands with pipes (`|`)

## Where to Go Next

### For Your Projects
- Learn Git (see our Git & GitHub course!)
- Set up development tools
- Deploy websites with Netlify or Vercel

### For Automation
- Learn bash scripting for repetitive tasks
- Set up aliases for your common commands
- Explore cron jobs for scheduled tasks

### For Fun
- Try a terminal-based text editor like `nano`
- Explore `tmux` for multiple terminal windows
- Play terminal games like `ninvaders` or `cowsay`

## Staying Safe

Remember:
- No "undo" in Terminal—be careful with `rm`
- Use `rm -i` when unsure
- Always `ls` before deleting

## Keep Practicing

The best way to learn is to use the Terminal for real tasks. Next time you need to find a file, try `find`. Next time you need to move files, try `mv` instead of dragging.

You'll be a Terminal pro in no time!
