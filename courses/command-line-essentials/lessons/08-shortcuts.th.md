# แป้นพิมพ์ลัด

ประหยัดเวลาด้วยแป้นพิมพ์ลัดที่จำเป็นเหล่านี้

## แป้นลัดการนำทาง

| แป้น | ทำอะไร |
|------|--------|
| `Ctrl + A` | ไปที่ต้นบรรทัด |
| `Ctrl + E` | ไปท้ายบรรทัด |
| `Ctrl + U` | ล้างทั้งบรรทัด |
| `Ctrl + C` | ยกเลิกคำสั่งปัจจุบัน |
| `Ctrl + L` | ล้างหน้าจอ (เหมือน `clear`) |

## แป้นลัดประวัติ

| แป้น | ทำอะไร |
|------|--------|
| `ลูกศรขึ้น` | คำสั่งก่อนหน้า |
| `ลูกศรลง` | คำสั่งถัดไป |
| `Ctrl + R` | ค้นหาประวัติคำสั่ง |
| `history` | แสดงคำสั่งก่อนหน้าทั้งหมด |

## เติมคำอัตโนมัติ

กด `Tab` เพื่อเติมคำอัตโนมัติ:
- ชื่อไฟล์
- ชื่อโฟลเดอร์
- คำสั่ง

กด `Tab` สองครั้งเพื่อดูตัวเลือกทั้งหมด

## คัดลอกและวาง

| แป้น | ทำอะไร |
|------|--------|
| `Ctrl + Shift + C` | คัดลอกข้อความที่เลือก |
| `Ctrl + Shift + V` | วางข้อความ |

หมายเหตุ: `Ctrl + C` และ `Ctrl + V` ปกติไม่ทำงานใน Terminal สำหรับคัดลอก/วาง

## ที่เป็นประโยชน์ที่สุด: `Ctrl + R`

กด `Ctrl + R` แล้วเริ่มพิมพ์เพื่อค้นหาประวัติคำสั่ง

ตัวอย่าง ถ้าคุณพิมพ์ `git commit` ก่อนหน้านี้ แค่:
- กด `Ctrl + R`
- พิมพ์ "git"
- กด Enter เพื่อรันอีกครั้ง

## ฝึกฝน

ลองทำตอนนี้:
1. พิมพ์อะไรบางอย่าง แล้ว `Ctrl + U` เพื่อล้าง
2. กด `ลูกศรขึ้น` เพื่อเอากลับมา
3. กด `Ctrl + L` เพื่อล้างหน้าจอ
