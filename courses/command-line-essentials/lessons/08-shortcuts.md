# Keyboard Shortcuts

Save time with these essential keyboard shortcuts.

## Navigation Shortcuts

| Shortcut | What it does |
|----------|--------------|
| `Ctrl + A` | Go to start of line |
| `Ctrl + E` | Go to end of line |
| `Ctrl + U` | Clear entire line |
| `Ctrl + C` | Cancel current command |
| `Ctrl + L` | Clear the screen (like `clear`) |

## History Shortcuts

| Shortcut | What it does |
|----------|--------------|
| `Up Arrow` | Previous command |
| `Down Arrow` | Next command |
| `Ctrl + R` | Search command history |
| `history` | Show all previous commands |

## Auto-Complete

Press `Tab` to auto-complete:
- File names
- Folder names
- Commands

Press `Tab` twice to see all options.

## Copy & Paste

| Shortcut | What it does |
|----------|--------------|
| `Ctrl + Shift + C` | Copy selected text |
| `Ctrl + Shift + V` | Paste text |

Note: Regular `Ctrl + C` and `Ctrl + V` don't work in Terminal for copy/paste.

## The Most Useful: `Ctrl + R`

Press `Ctrl + R` and start typing to search your command history.

For example, if you typed `git commit` earlier, just:
- Press `Ctrl + R`
- Type "git"
- Press Enter to run it again

## Practice Makes Perfect

Try these now:
1. Type something, then `Ctrl + U` to clear
2. Press `Up Arrow` to get it back
3. Press `Ctrl + L` to clear the screen
