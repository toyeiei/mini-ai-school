# Your First Commands

Time to type your first commands. Let's start with three simple ones.

## The `echo` Command

`echo` repeats what you say. Like a parrot:

```bash
echo "Hello, world!"
```

Output:
```
Hello, world!
```

**Real-life use:** Testing if your Terminal is working, or printing messages in scripts.

## The `date` Command

`date` shows the current date and time:

```bash
date
```

Output:
```
Thu Apr  3 12:30:00 +07 2025
```

**Real-life use:** Checking timestamps, adding dates to backup files.

## The `whoami` Command

`whoami` asks "who am I logged in as?"

```bash
whoami
```

Output:
```
kasidis
```

**Real-life use:** Confirming you're the right user before doing something important.

## Practice

Try typing these one by one:

1. `echo "My name is..."`
2. `date`
3. `whoami`

Notice how each command finishes quickly? The Terminal is fast because it doesn't need to load buttons or graphics. It just does exactly what you ask.

## What About Errors?

If you type something wrong, you'll see an error:

```bash
ech0 "Oops"
```

Output:
```
-bash: ech0: command not found
```

The computer literally says: "I don't know that command." This is normal when learning. Just try again with the correct spelling.
