# ขั้นตอนถัดไป

ยินดีด้วย! คุณเรียนรู้พื้นฐาน Terminal แล้ว

## สิ่งที่คุณรู้ตอนนี้

- นำทางโฟลเดอร์ (`pwd`, `ls`, `cd`)
- สร้างและลบไฟล์ (`touch`, `mkdir`, `rm`)
- ดูเนื้อหาไฟล์ (`cat`, `head`, `tail`, `less`)
- คัดลอกและย้ายไฟล์ (`cp`, `mv`)
- ค้นหาไฟล์และเนื้อหา (`find`, `grep`)
- ใช้แป้นพิมพ์ลัด (`Ctrl + C`, `Tab` ฯลฯ)
- รวมคำสั่งด้วย pipes (`|`)

## ไปต่อที่ไหน

### สำหรับโปรเจกต์ของคุณ
- เรียน Git (ดูคอร์ส Git & GitHub ของเรา!)
- ตั้งค่าเครื่องมือสำหรับนักพัฒนา
- Deploy เว็บไซต์ด้วย Netlify หรือ Vercel

### สำหรับการทำงานอัตโนมัติ
- เรียน bash scripting สำหรับงานซ้ำๆ
- ตั้งค่า aliases สำหรับคำสั่งที่ใช้บ่อย
- ลอง cron jobs สำหรับงานตามกำหนดเวลา

### เพื่อความสนุก
- ลอง text editor บน Terminal เช่น `nano`
- สำรวจ `tmux` สำหรับหน้าต่าง Terminal หลายอัน
- เล่นเกมบน Terminal เช่น `ninvaders` หรือ `cowsay`

## การใช้งานอย่างปลอดภัย

จำไว้:
- ไม่มี "ย้อนกลับ" ใน Terminal—ระวังกับ `rm`
- ใช้ `rm -i` เมื่อไม่แน่ใจ
- พิมพ์ `ls` ก่อนลบเสมอ

## ฝึกฝนต่อไป

วิธีเรียนรู้ที่ดีที่สุดคือใช้ Terminal สำหรับงานจริง ครั้งหน้าที่ต้องหาไฟล์ ลอง `find` ครั้งหน้าที่ต้องย้ายไฟล์ ลอง `mv` แทนการลากวาง

คุณจะเป็น Terminal pro ไม่นาน!
