# ดูเนื้อหาไฟล์

คุณสามารถอ่านไฟล์ใน Terminal ได้เลย ไม่ต้องเปิดแอปอื่น

## ดูเร็ว: `head` และ `tail`

`head` แสดงบรรทัดแรกๆ ของไฟล์:

```bash
head readme.txt
```

`tail` แสดงบรรทัดสุดท้าย:

```bash
tail readme.txt
```

**ใช้จริง:** ดูส่วนต้น/ท้ายของไฟล์บันทึกเพื่อดูกิจกรรมล่าสุด

## อ่านทั้งหมด: `cat`

`cat` (concatenate) แสดงเนื้อหาทั้งไฟล์:

```bash
cat readme.txt
```

นี่พิมพ์ทุกอย่างบนหน้าจอ

**ใช้จริง:** อ่านไฟล์สั้นๆ เช่นไฟล์ตั้งค่าหรือไฟล์ข้อความเล็ก

## อ่านทีละหน้า: `less`

สำหรับไฟล์ยาว ใช้ `less`:

```bash
less long-file.txt
```

- กด `Space` เพื่อเลื่อนลง
- กด `b` เพื่อเลื่อนขึ้น
- กด `q` เพื่อออก

นี่เป็นประโยชน์สำหรับอ่านเอกสารหรือไฟล์ใหญ่

## นับคำ: `wc`

`wc` นับบรรทัด คำ และตัวอักษร:

```bash
wc readme.txt
```

ผลลัพธ์:
```
  42  156  892 readme.txt
```

หมายความ: 42 บรรทัด, 156 คำ, 892 ตัวอักษร

**ใช้จริง:** ตรวจสอบความยาวเอกสารก่อนแชร์

## สรุป

| คำสั่ง | ทำอะไร |
|--------|--------|
| `head ไฟล์` | 10 บรรทัดแรก |
| `tail ไฟล์` | 10 บรรทัดสุดท้าย |
| `cat ไฟล์` | ทั้งไฟล์ |
| `less ไฟล์` | อ่านทีละหน้า |
| `wc ไฟล์` | นับบรรทัด/คำ |
