# HTML คืออะไร

HTML (HyperText Markup Language) คือภาษาของเว็บ มันกำหนดโครงสร้างของหน้าเว็บ

## HTML = เนื้อหา + Markup

- **เนื้อหา**: ข้อความ, รูปภาพ, วิดีโอ
- **Markup**: แท็กที่อธิบายเนื้อหา

## หน้า HTML แรกของคุณ

```html
<!DOCTYPE html>
<html>
<head>
    <title>หน้าแรกของฉัน</title>
</head>
<body>
    <h1>สวัสดีชาวโลก!</h1>
    <p>นี่คือหน้าเว็บแรกของฉัน</p>
</body>
</html>
```

## ส่วนสำคัญ

- `<!DOCTYPE html>` - บอกเบราว์เซอร์ว่าเป็นหน้า HTML5
- `<html>` - องค์ประกอบราก ประกอบด้วยทุกอย่าง
- `<head>` - ข้อมูลเมตา (ชื่อ, styles)
- `<body>` - เนื้อหาที่มองเห็น

## วิธีดู

1. บันทึกเป็น `index.html`
2. เปิดในเบราว์เซอร์
3. เห็นหน้าเว็บของคุณ!

## ทำไม HTML ถึงสำคัญ

ทุกเว็บไซต์ใช้ HTML มันคือพื้นฐานของการพัฒนาเว็บ เรียนอันนี้ก่อน ส่วนที่เหลือจะเข้าใจง่ายขึ้น

## สรุป

- HTML กำหนดโครงสร้างเนื้อหาเว็บ
- ใช้แท็กเช่น `<h1>`, `<p>`, `<div>`
- บันทึกเป็นไฟล์ `.html`
- เบราว์เซอร์แสดงผลเป็นภาพ
