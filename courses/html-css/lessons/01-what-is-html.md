# What is HTML

HTML (HyperText Markup Language) is the language of the web. It defines the structure of web pages.

## HTML = Content + Markup

- **Content**: Text, images, videos
- **Markup**: Tags that describe the content

## Your First HTML Page

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>
```

## Key Parts

- `<!DOCTYPE html>` - Tells browser this is an HTML5 page
- `<html>` - Root element, contains everything
- `<head>` - Meta info (title, styles)
- `<body>` - Visible content

## How to View

1. Save as `index.html`
2. Open in any browser
3. See your page!

## Why HTML Matters

Every website uses HTML. It's the foundation of web development. Learn this first, and everything else makes more sense.

## Summary

- HTML structures web content
- Uses tags like `<h1>`, `<p>`, `<div>`
- Saved as `.html` files
- Browser renders it visually
